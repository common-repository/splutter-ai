Integrate Splutter AI Chatbot into your WordPress site effortlessly.

== Description ==

**Splutter AI Chatbot** allows you to seamlessly add a fully customizable, AI-driven chatbot to your WordPress site. With hot-swappable models, real-time integrations, voice tools, automated bookings, and lead capture capabilities, this plugin delivers a highly interactive and efficient user experience.

This Plugin requires a Splutter AI Chatbot ID to function. You can obtain your Chatbot ID by logging into your Splutter AI, creating an AI Chatbot or Choose an Existing Chatbot, and navigating to the chatbot dashboard, clicking layout, and copying the ID.

Complete Customization:
- Layout: Fully customizable layout with options for color, gradients, opacity, blur effects, and even animations to fit your brand's aesthetic.
- Prompt Engineering Behavior Tab: Fine-tune your chatbot’s conversational flow with prompt engineering settings.
- Hot-Swappable: Instantly change models, tools, and integrations without downtime.
- Real-Time Metrics & Debugging: Track chatbot performance, user interactions, and conversion rates.
- Chat History: Store and review previous conversations for insight and analytics.
- Notifications: Enable email and SMS alerts for key chatbot events.
- Bot Debugger: Analyze and troubleshoot the chatbot’s behavior and responses.

Key Features:
- Calendar Tool: Automate bookings and schedule management.
- Ticketing Tool: Create, assign, and resolve support tickets directly through the chatbot.
- Lead Capture Tool: Automate lead generation and follow-ups with integration into your CRM.
- Voice Tool: Enable voice-based interactions with real-time conversations.
- Security Tool: Enhance your chatbot’s security with customizable LLM security options.
- Recommender Tool: Personalize user experiences by offering recommendations based on their interactions.
- API Bundle Tool: Integrate API calls and webhooks for dynamic, real-time data exchanges.



== Installation ==

1. **Download the Plugin:**
   - A. Install on the Official Wordpress Website: Splutter AI Chatbot
   - B. Click [here](https://www.splutter.ai/wordpress/splutter-ai.zip) to download the `splutter-ai.zip` file.

2. **Install the Plugin:**
   A. If via Official Wordpress Plugin its installed, simply activate the plugin.
   B. If via zip file, follow these steps:
      - Log in to your WordPress dashboard.
      - Navigate to **Plugins > Add New**.
      - Click on the **Upload Plugin** button.
      - Choose the `splutter-chatbot.zip` file you downloaded.
      - Click **Install Now**.

3. **Activate the Plugin:**

   - After installation, click on **Activate Plugin**.

4. **Configure the Plugin Settings:**

   - Go to **Splutter AI Chatbot > Splutter AI Chatbot Settings** in the admin menu.
   - Enter your **Chatbot ID** provided by Splutter AI.
   - Adjust the settings as per your preference:
     - **Alignment:** Choose from predefined positions or select "Custom."
     - **Size:** Set a value between 0.2 and 10 (default is 0.5).
     - **Background Color:** Use the color picker to select a color.
     - **Blur Effect:** Set a value between 0 and 100 (default is 10).
     - **Draggable:** Enable or disable the draggable option.
   - Click **Save Changes**.

5. **Verify the Chatbot:**

   - Visit your website to ensure the chatbot appears and functions correctly.

== Frequently Asked Questions ==

= Where do I get my Chatbot ID? =

You can obtain your Chatbot ID by logging into your Splutter AI account and navigating to the chatbot dashboard, clicking layout, and copying the ID.

= Can I customize the chatbot's appearance? =

Yes, you can customize the alignment, size, background color, blur effect, and draggable option through the Splutter AI Chatbot Dashboard.

= Is the chatbot responsive? =

Absolutely! The chatbot is designed to work seamlessly across all devices and screen sizes.

= How do I update the plugin? =

You can update the plugin by downloading the latest version and installing it via the WordPress dashboard, or by updating it directly if available.
